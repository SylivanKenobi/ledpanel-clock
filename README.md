# Ledpanel Clock

This clock uses an [NeoPixel 8x32](https://www.bastelgarage.ch/neopixel-flexmatrix-8x32-256-x-ws2812b-rgb-led?search=8x32) ledpanel and a Raspberry Pi Pico.

An example Project for the [Neopixel Panel library](https://github.com/SylivanKenobi/pico-neopixel-matrix)
